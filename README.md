\# Mushroom Yield Predictor

Zelbytes Agritech Internship — AI Data Analyst



\## Problem Statement

Predict daily mushroom yield (kg) in a climate-controlled

polyhouse using sensor readings: temperature (°C),

relative humidity (%), and CO2 (ppm).
